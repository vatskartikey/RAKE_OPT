BOXN LOGISTIC PLANNING TOOLKIT

Version 1.0

The tool provides a decision support system to allocate rakes at region/siding based on position of empty rakes unloading location of loaded rakes others,indented forecasted demand considering its priority criticality,age and probability to get loaded near the destination.The model provides custom controls to user for prioritizing demands and editing rakes availability.The model takes in the capacities specified in the system for allocations and tries to reduce congestion on the network.The capacity can be viewed by pressing the button on bottom left.The capacity can be edited by regional users based on dynamicity in field.The RB operator can edit the control settings to specify parameters controlling the assignment.
